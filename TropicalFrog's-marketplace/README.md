# TropicalFrog's Seanime Extension Marketplace

Welcome to the **TropicalFrog Extension Marketplace**! This is a curated collection of online streaming extensions for **[Seanime](https://github.com/5rahim/seanime)**, providing high-quality anime, hentai, and multi-language streams directly in your Seanime dashboard.

---

## 🚀 Quick Installation

To add all these extensions to your Seanime client at once, follow these simple steps:

1. Copy this marketplace URL:
   ```text
   https://raw.githubusercontent.com/TropicalFrog3/seanime-extension/refs/heads/main/TropicalFrog's-marketplace/main.json
   ```
2. Open **Seanime** and navigate to **Extensions** -> **Marketplaces**.
3. Click **Add External Marketplace**, paste the URL, and click **Confirm**.
4. Install your desired extensions with a single click!
